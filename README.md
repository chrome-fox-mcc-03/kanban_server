# Catban server

**Base URL: http://localhost:3000**


## User
---

*Login*
---

  Returns token and username of the user with status of 200.

* **URL**

  /login

* **Method:**

  `GET`

* **Data Params**

  * `email=[string]`
  * `password=[string]`

* **Success Response:**

  * **Code:** 200 <br />
    **Content:** `{ token, username }`
 
* **Error Response:**

  * **Code:** 401 BAD REQUEST <br />
    **Content:** `{ errors : "Wrong email / password" }`

* **Sample Call:**

  ```javascript
    Axios({
      url: 'http://localhost:3000/login',
      method : "GET",
      data: {
        email: ...,
        password: ...
      }
    })
      .then(({data}) => {
        ...
      })
      .catch(err => {
        ...
      })
  ```
